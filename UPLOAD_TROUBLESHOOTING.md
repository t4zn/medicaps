# Upload Troubleshooting Guide

## File Upload Issues

### 413 Error - File Too Large

If you're getting a "413 Payload Too Large" error when uploading files, especially from mobile devices:

**Solutions:**
1. **Check file size**: Ensure your PDF is under 50MB
2. **Compress your PDF**: Use online tools like SmallPDF, ILovePDF, or PDF24 to reduce file size
3. **Split large files**: If your PDF has many pages, consider splitting it into smaller parts
4. **Use desktop**: Mobile uploads can be more sensitive to large files due to network constraints

### Mobile Upload Tips

**For better mobile upload experience:**
- Use WiFi instead of mobile data for large files
- Close other apps to free up memory
- Ensure stable internet connection
- Try uploading during off-peak hours

### Supported File Types

- **Accepted**: PDF files only
- **Maximum size**: 50MB per file
- **Recommended size**: Under 10MB for best performance

### Common Error Messages

- **"File too large"**: Your file exceeds 50MB limit
- **"Upload timeout"**: Network connection issue or file too large
- **"Only PDF files allowed"**: You selected a non-PDF file
- **"Connection failed"**: Check your internet connection

### Still Having Issues?

If you continue to experience upload problems:
1. Try a different browser
2. Clear your browser cache
3. Disable browser extensions temporarily
4. Contact support with details about your file size and device type
